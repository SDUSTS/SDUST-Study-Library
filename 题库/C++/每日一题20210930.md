# 题目
现在输入一个数字forn（n<=10）,请你输出一个直角边长度为n的数字三角形，一个长度是由一个两位数构成的，如果数字没有两位（个位数）则在前面加上前导零（例如 1应为01）三角形中的数字要求从左到右顺次排列，下一行的第一个数字比上一行的最后一个数字大一。

## 样例：
~~~
样例输入：3
样例输出：
010203
0405
06
~~~
~~~
样例输入：5
样例输出：
0102030405
06070809
101112
1314
15
~~~

### 代码参考
```c++
#include<iostream>
#include<stdio.h>  
using namespace std;
int main()
{
	int n,t=0,a;	
	cin>>n;
	a=n;			/
	for(int i=0;i<n;i++)
    {   //n行
		for(int j=0;j<a;j++)
		{   //每行a列
			t++;
			printf("%02d",t);
		}
		a--;	  	//每输出一行,减少一列
		cout<<endl;
	}
	return 0;
}
```
