# 指南

每一天的习题有两个或多个文件夹。

其中习题的名称以日期命令。每一个习题分为.md文件和其他文件。

其它文件如类似.c文件.py文件.m文件。这些文件是包含题目的具体代码和注释。

**注意：** 有些习题的参考代码就在.md文件中。
